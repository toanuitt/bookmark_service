# bookmark_service

A lightweight Go service for managing bookmarks with secure password handling and health monitoring capabilities.

## Features

- RESTful API for bookmark management
- Built-in password service for secure credential storage
- Health check endpoint for service monitoring
- Production-ready with comprehensive test coverage

## Installation

### Build from source

```bash
# Clone the repository
git clone https://github.com/toanuitt/bookmark_service.git
cd bookmark_service

# Run
make run

```
